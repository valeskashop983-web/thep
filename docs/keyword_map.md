# Keyword Map

## Purpose

This file prioritizes categories, routes, and editorial topics. Update it from Search Console, permitted keyword tools, manual SERP review, and internal conversion data.

## Priority Clusters

| Cluster | Intent | Route | Notes |
| --- | --- | --- | --- |
| Telegram channels | Directory discovery | `/channels`, `/c/[slug]` | Broad navigational and discovery intent. |
| Telegram groups | Community discovery | `/groups`, `/c/telegram-groups` | Strong join intent. |
| Telegram bots | Tool discovery | `/bots`, `/c/telegram-bots` | Good fit for AI/SaaS affiliates. |
| AI girlfriend | AI companion discovery | `/ai-girlfriend`, `/c/ai-girlfriend` | Requires careful adult-content and disclosure policies. |
| NSFW Telegram | Adult discovery | `/c/nsfw-telegram` | Needs age gating, moderation, and strict illegal-content exclusion. |
| Creator channels | Creator discovery | `/creators`, `/c/creators` | Secondary focus. |

## Category Creation Rules

- Create indexable categories only when there is enough inventory and distinct search intent.
- Keep thin or speculative categories as non-indexable until they have useful listings.
- Merge synonyms into one canonical category.
- Use tags for modifiers that do not deserve standalone landing pages yet.

## Data To Track

- Query.
- Search intent.
- Target route.
- Volume estimate.
- Keyword difficulty estimate.
- Current ranking URL.
- Click-through rate.
- Conversion value.
- Required content improvements.

## Update Cadence

- Weekly during launch.
- Monthly after traffic stabilizes.
- Immediately after major algorithm, policy, or monetization changes.
